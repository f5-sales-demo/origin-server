## आवश्यक

### Azure सदस्यता

एक सक्रिय Azure सदस्यता जिसमें निम्नलिखित बनाने की अनुमति हो:

- संसाधन समूह
- वर्चुअल नेटवर्क और सबनेट
- नेटवर्क सुरक्षा समूह
- सार्वजनिक IP पते
- वर्चुअल मशीनें (Standard_D16s_v3 -- 41-कंटेनर कार्यभार के लिए 16 vCPU, 64 GiB RAM)

### Azure CLI

Azure CLI को स्थापित करें और प्रमाणित करें:

```bash
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

az login

az account show --query '{name:name, id:id}' -o table
```

### Terraform

Terraform >= 1.5 आवश्यक है:

```bash
terraform version
```

यदि स्थापित नहीं है, तो HashiCorp APT रिपॉजिटरी का उपयोग करें:

```bash
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
```

### Azure AD अनुमतियाँ

आपके Azure AD खाते में `User.Read` अनुमतियाँ होनी चाहिए। परिनियोजक पहचानकर्ता (संसाधन नामकरण में उपयोग किया जाता है) आपकी Azure AD प्रोफ़ाइल से स्वतः व्युत्पन्न होता है। सेवा प्रिंसिपल या प्रबंधित पहचान प्रमाणीकरण के लिए, `deployer` Terraform चर को स्पष्ट रूप से सेट करें।

### SSH कुंजी युगल

VM पहुँच के लिए एक SSH कुंजी युगल:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/origin-server-key -N ""
```

## बनाए गए Azure संसाधन

| संसाधन | SKU |
|---|---|
| Ubuntu 24.04 VM | Standard_D16s_v3 (16 vCPU, 64 GiB) |
| सार्वजनिक IP | Standard, स्थिर |
| OS डिस्क | 60 GiB Premium SSD |
| VNet + NSG | डिफ़ॉल्ट |

वर्तमान लागत के लिए [Azure मूल्य निर्धारण कैलकुलेटर](https://azure.microsoft.com/en-us/pricing/calculator/) देखें। शुल्क रोकने के लिए जब लैब उपयोग में न हो तब `terraform destroy` चलाएँ। प्रक्रिया के लिए [Teardown](../07-teardown/) देखें।

:::note
41-कंटेनर परिनियोजन के लिए Standard_D16s_v3 (64 GiB RAM) आवश्यक है। कंटेनर मेमोरी आवंटन: 4 Juice Shop इंस्टेंस प्रत्येक 1 GiB (4 GiB), 4 VAmPI प्रत्येक 512 MiB (2 GiB), 4 DVWA-FPM + MariaDB (1.8 GiB), 4 httpbin प्रत्येक 256 MiB (1 GiB), 4 CSD Demo प्रत्येक 256 MiB (1 GiB), 4 whoami प्रत्येक 64 MiB (256 MiB), 4 DVGA प्रत्येक 256 MiB (1 GiB), 4 RESTaurant + PostgreSQL (1.5 GiB), crAPI 7 माइक्रोसर्विसेज़ (2 GiB), साथ ही nginx, OS और बफ़र कैश। कुल कंटेनर आवंटन लगभग 14.5 GiB (64 GiB का 22.7%) है।
:::

:::note
NSG पोर्ट 80 (HTTP) और 8888 (crAPI) पर इनबाउंड ट्रैफ़िक की अनुमति देता है। पोर्ट 8888 आवश्यक है क्योंकि crAPI एक सिंगल-पेज एप्लिकेशन है जो अपने API पथों को हार्डकोड करती है और इसे पथ उपसर्ग के पीछे परोसा नहीं जा सकता।
:::
