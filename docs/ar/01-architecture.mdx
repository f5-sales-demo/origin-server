## الغرض

يوفر هذا المكون خادم أصل واحد يستضيف تطبيقات ويب متعددة ذات ثغرات أمنية لعروض اختبار الأمان. وهو يمثل "الأصل" في بنية موازن الأحمال النموذجية -- خادم المحتوى الخلفي الذي يحميه موازن أحمال F5 XC HTTP.

في بنى الإنتاج:

```
End User -> F5 XC HTTP LB (WAF/Bot/API Security) -> Origin Server -> Application
```

يستبدل هذا المكون خادم تطبيق إنتاج حقيقي بجهاز افتراضي مصمم خصيصاً يشغل تطبيقات معروفة ذات ثغرات أمنية تُفعّل قواعد WAF وسياسات أمان API واكتشاف الروبوتات.

## البنية المعمارية

```mermaid
graph LR
    Client[Client Browser] --> XCHLB[F5 XC HTTP LB]
    XCHLB --> NGINX[nginx Reverse Proxy<br/>Ubuntu 24.04 · D16s_v3<br/>reuseport · 16 workers]
    NGINX --> |/| DEFAULT[Landing Page]
    NGINX --> |/juice-shop/| JS[Juice Shop ×4<br/>Ports 3001-3004<br/>hash cookie_token · cache]
    NGINX --> |/dvwa/| DVWA[DVWA-FPM ×4<br/>Ports 8101-8104<br/>hash cookie_PHPSESSID]
    NGINX --> |/vampi/| VAMPI[VAmPI ×4<br/>Ports 5101-5104<br/>ip_hash · gunicorn]
    NGINX --> |/httpbin/| HTTPBIN[httpbin ×4<br/>Ports 8201-8204<br/>gunicorn -w 4]
    NGINX --> |/whoami/| WHOAMI[whoami ×4<br/>Ports 8082-8085]
    NGINX --> |/csd-demo/| CSD[CSD Demo ×4<br/>Ports 5001-5004<br/>ip_hash · gunicorn -w 1]
    NGINX --> |/dvga/| DVGA[DVGA ×4<br/>Ports 5201-5204<br/>ip_hash · SQLite]
    NGINX --> |/restaurant/| REST[RESTaurant ×4<br/>Ports 8301-8304<br/>round-robin]
    XCHLB --> |:8888| CRAPI[crAPI ×7 microservices<br/>Port 8888<br/>web · identity · community<br/>workshop · postgres · mongo · mailhog]
    DVWA --> DB[(MariaDB 10.11<br/>dvwa-db)]
    REST --> RESTDB[(PostgreSQL 15.4<br/>restaurant-db)]
```

**41 حاوية** على جهاز افتراضي Standard_D16s_v3 (16 وحدة معالجة مركزية افتراضية، 64 جيبي بايت ذاكرة، 60 جيبي بايت قرص).

الوكيل العكسي nginx:

- **يستمع على المنفذ 80** مع `reuseport` و `backlog=4096` لحركة مرور CDN عالية التزامن
- **يوجه حسب بادئة المسار** إلى مجموعات upstream متوازنة الأحمال (4 نسخ لكل تطبيق)
- **الجلسات الثابتة** تمنع فقدان الحالة: `hash $cookie_token` لـ Juice Shop، `hash $cookie_PHPSESSID` لـ DVWA، `ip_hash` لـ VAmPI و CSD Demo (حالة SQLite/في الذاكرة لكل نسخة)
- **ذاكرة التخزين المؤقت للوكيل** للأصول الثابتة في Juice Shop (منطقة 10 ميجابايت، حد أقصى 100 ميجابايت، مدة صلاحية 60 ثانية)
- **تسجيل الوصول معطل** لمنع استنفاد القرص أثناء اختبار حمل CDN (logrotate كدفاع متعدد الطبقات)
- **يمرر رؤوس العميل** (`X-Real-IP`، `X-Forwarded-For`، `X-Forwarded-Proto`) لرؤية الأصل
- **ضبط النواة** عبر sysctl: `somaxconn=65535`، `tcp_tw_reuse=1`، `ip_local_port_range=1024-65535`

## تصميم المكونات المعيارية

هذا جزء واحد من بيئة مختبر أكبر. كل مكون مستقل بذاته ويتم نشره بشكل مستقل:

- **هذا المكون** يوفر خادم الأصل (nginx + حاويات Docker على جهاز Azure الافتراضي)
- **محاكي CDN** يوفر طبقة حافة CDN (تخزين مؤقت nginx على جهاز Azure الافتراضي)
- **مكونات أخرى** توفر تكوين F5 XC وDNS وسياسات WAF وأمان API وغيرها

يضيف المشغل البشري المكونات واحداً تلو الآخر. وثائق كل مكون مكتوبة بحيث يمكن لمساعد ذكاء اصطناعي قراءتها ونشر البنية التحتية بشكل مستقل.

## لماذا هذه التطبيقات

| التطبيق | سبب الاختيار |
|---|---|
| **Juice Shop** | المشروع الرائد لـ OWASP؛ تطبيق صفحة واحدة حديث بـ Node.js يحتوي على أكثر من 100 تحدي يغطي OWASP Top 10؛ يتم صيانته بنشاط؛ 4 نسخ مع ذاكرة تخزين مؤقت للوكيل |
| **DVWA** | معيار صناعي لاختبار WAF؛ مستويات أمان قابلة للتعديل (منخفض/متوسط/عالي/مستحيل)؛ بناء مخصص بـ php-fpm + nginx للأداء؛ واجهة خلفية MariaDB 10.11 مشتركة |
| **VAmPI** | مصمم خصيصاً لـ OWASP API Security Top 10؛ واجهة REST API بثغرات معروفة؛ gunicorn مع 4 عمال لكل نسخة؛ ip_hash ثابت لاتساق SQLite |
| **httpbin** | خدمة اختبار HTTP المرجعية من Kenneth Reitz؛ gunicorn مع 4 عمال gevent؛ مفيدة لعروض API وفحص الطلبات |
| **whoami** | خادم صدى الطلبات من Traefik؛ يعرض تفاصيل الطلب الكاملة كما يراها الأصل -- ضروري للتحقق من حقن رؤوس F5 XC |
| **CSD Demo** | صفحة دفع مخصصة مع 5 هجمات بنمط Magecart قابلة للتبديل (سارق بطاقات، مخترق نماذج، راصد لوحة مفاتيح، معدّن عملات مشفرة، اختطاف DOM)؛ نقطة نهاية تسريب + لوحة تحكم المهاجم؛ gunicorn بعامل واحد للحفاظ على حالة الذاكرة |
| **DVGA** | تطبيق GraphQL ذو الثغرات الأمنية؛ ثغرات خاصة بـ GraphQL تشمل الحقن وحجب الخدمة وهجمات التجميع وتجاوز المصادقة؛ واجهة GraphiQL للاستكشاف التفاعلي؛ ip_hash ثابت لـ SQLite لكل نسخة |
| **RESTaurant** | لعبة واجهة RESTaurant API ذات الثغرات الأمنية؛ مصمم خصيصاً لـ OWASP API Security Top 10 2023؛ FastAPI مع واجهة Swagger؛ واجهة خلفية PostgreSQL 15.4 مشتركة؛ يغطي BOLA وBFLA والتعيين الجماعي وSSRF والحقن |
| **crAPI** | واجهة API السخيفة تماماً من OWASP؛ بنية من 7 خدمات مصغرة تغطي BOLA وBFLA والتعيين الجماعي وSSRF والتلاعب بـ JWT وحقن NoSQL؛ منفذ مخصص 8888 (تطبيق صفحة واحدة بمسارات API مضمنة)؛ MailHog لالتقاط البريد الإلكتروني |
