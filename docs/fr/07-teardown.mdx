Lorsque la session de lab est terminée, détruisez toutes les ressources Azure pour arrêter la facturation.

## Terraform Destroy

Depuis le répertoire contenant la configuration Terraform :

```bash
terraform plan -destroy

terraform destroy
```

Terraform demandera une confirmation. Tapez `yes` pour continuer.

## Vérifier le nettoyage

Confirmez que toutes les ressources ont été supprimées :

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Nettoyer l'état local

Supprimez l'état Terraform et les fichiers mis en cache :

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Nettoyage F5 XC

Si vous avez créé des ressources F5 XC pointant vers ce serveur d'origine, supprimez-les :

1. Supprimez le load balancer HTTP
2. Supprimez l'origin pool
3. Supprimez toutes les politiques WAF créées pour les tests
