Al termine della sessione di laboratorio, distruggere tutte le risorse Azure per interrompere la fatturazione.

## Terraform Destroy

Dalla directory contenente la configurazione Terraform:

```bash
terraform plan -destroy

terraform destroy
```

Terraform chiederà conferma. Digitare `yes` per procedere.

## Verifica della pulizia

Confermare che tutte le risorse siano state rimosse:

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Pulizia dello stato locale

Rimuovere lo stato Terraform e i file memorizzati nella cache:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Pulizia F5 XC

Se sono state create risorse F5 XC che puntano a questo server di origine, rimuoverle:

1. Eliminare il bilanciatore del carico HTTP
2. Eliminare il pool di origine
3. Eliminare eventuali policy WAF create a scopo di test
