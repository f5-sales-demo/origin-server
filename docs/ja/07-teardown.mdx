ラボセッションが完了したら、課金を停止するためにすべてのAzureリソースを削除します。

## Terraform Destroy

Terraform設定ファイルが含まれるディレクトリから実行します：

```bash
terraform plan -destroy

terraform destroy
```

Terraformから確認を求められます。`yes` と入力して続行してください。

## クリーンアップの確認

すべてのリソースが削除されたことを確認します：

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## ローカルステートの削除

Terraformのステートファイルとキャッシュファイルを削除します：

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## F5 XCのクリーンアップ

このオリジンサーバーを参照するF5 XCリソースを作成した場合は、それらを削除してください：

1. HTTPロードバランサーを削除する
2. オリジンプールを削除する
3. テスト用に作成したWAFポリシーを削除する
