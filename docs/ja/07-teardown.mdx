ラボセッションが完了したら、課金を停止するために全ての Azure リソースを削除します。

## Terraform による削除

Terraform 設定ファイルが格納されているディレクトリから実行します：

```bash
terraform plan -destroy

terraform destroy
```

Terraform が確認を求めます。`yes` と入力して続行します。

## クリーンアップの確認

全てのリソースが削除されたことを確認します：

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## ローカル状態のクリーンアップ

Terraform の状態ファイルとキャッシュファイルを削除します：

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## F5 XC のクリーンアップ

このオリジンサーバーを指す F5 XC リソースを作成した場合は、それらを削除します：

1. HTTP ロードバランサーを削除する
2. オリジンプールを削除する
3. テスト用に作成した WAF ポリシーを削除する
