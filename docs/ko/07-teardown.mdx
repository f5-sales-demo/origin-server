실습 세션이 완료되면 과금을 중지하기 위해 모든 Azure 리소스를 삭제합니다.

## Terraform Destroy

Terraform 구성 파일이 있는 디렉터리에서 실행합니다:

```bash
terraform plan -destroy

terraform destroy
```

Terraform이 확인을 요청합니다. 계속 진행하려면 `yes`를 입력하세요.

## 정리 확인

모든 리소스가 제거되었는지 확인합니다:

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## 로컬 상태 정리

Terraform 상태 및 캐시 파일을 제거합니다:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## F5 XC 정리

이 오리진 서버를 가리키는 F5 XC 리소스를 생성한 경우, 해당 리소스를 제거합니다:

1. HTTP 로드 밸런서 삭제
2. 오리진 풀 삭제
3. 테스트를 위해 생성한 WAF 정책 삭제
