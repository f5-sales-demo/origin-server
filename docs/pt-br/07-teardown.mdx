Quando a sessão do laboratório estiver concluída, destrua todos os recursos Azure para interromper a cobrança.

## Terraform Destroy

A partir do diretório que contém a configuração do Terraform:

```bash
terraform plan -destroy

terraform destroy
```

O Terraform solicitará confirmação. Digite `yes` para prosseguir.

## Verificar Limpeza

Confirme que todos os recursos foram removidos:

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Limpar Estado Local

Remova o estado do Terraform e os arquivos em cache:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Limpeza do F5 XC

Se você criou recursos F5 XC apontando para este servidor de origem, remova-os:

1. Exclua o HTTP load balancer
2. Exclua o origin pool
3. Exclua quaisquer políticas WAF criadas para testes
