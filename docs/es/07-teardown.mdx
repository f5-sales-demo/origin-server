Cuando la sesión del laboratorio esté completa, destruye todos los recursos de Azure para detener la facturación.

## Destrucción con Terraform

Desde el directorio que contiene la configuración de Terraform:

```bash
terraform plan -destroy

terraform destroy
```

Terraform solicitará confirmación. Escribe `yes` para proceder.

## Verificar la Limpieza

Confirma que todos los recursos han sido eliminados:

```bash
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Limpiar el Estado Local

Elimina el estado de Terraform y los archivos en caché:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Limpieza de F5 XC

Si creaste recursos de F5 XC apuntando a este servidor de origen, elimínalos:

1. Elimina el HTTP load balancer
2. Elimina el origin pool
3. Elimina cualquier política WAF creada para pruebas
